# 💳 Bank Account Management System

A simple Java project built to demonstrate the use of **packages** and **object-oriented programming** by implementing basic banking operations.

## ✨ Features
- Create bank accounts
- Deposit money
- Withdraw money
- Transfer funds
- Modular package-based structure

## 📦 Packages
- `bank.models` – Account & Customer classes
- `bank.operations` – Deposit, Withdraw & Transfer logic
- `bank.ui` – Main application interface

## 🛠️ Tech Stack
- Java
- OOP
- Java Packages

## 🎯 Objective
To understand how Java packages organize code, improve modularity, and promote code reuse through a simple banking application.
